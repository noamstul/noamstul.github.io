The Joys of Bootstrap

Bootstrap is awesome! It’s an open-source web development toolkit that allows you to take all the fuss out of developing websites and apps. Say goodbye to CSS and Javascipt, and enjoy all the handy pre-built components that Bootstrap has to offer. From navigation bars, to helpful layout tools, and even cards and built in colours, Bootstrap has it all.

Getting started

Are you ready for all the greatness that bootstrap has to offer? Getting your first bootstrap site started is just as easy as you think. All you have to do it go to getbootstrap.com, and you’ll find a few easy ways to get your html document bootstrapped and ready to go. The easiest way, in my opinion, is to copy the starter template down below.
Starter Instructions
  • 1 Head to getbootstrap.com and click "Get started"
  • 2 Scroll down to "Starter template"
  • 3 Copy the HTML in the grey window
  • 4 Paste it in your blank HTML document. You're ready to start!

Cards

Okay so what exactly are we talking about with cards? Cards can be used in many different ways, but if you want an easy example, take a look at our topics up at the top of the page. See how nicely contained each box is? well you can do that and more with the card component.

There are many different ways you can make a card with Bootstrap, but the thing you need to know is that they mainly consist of three parts: the card header, card body, and card footer. Any of these is optional, but you'll probably want to at least have a card body, and most of the time a card head.

Forms

Here's something that you'll inevitably have to do in any intro to responsive web design course: Forms! Whether you're designing a site for yourself or a client, you'll find that you will eventually need to create a form. It can be a real pain to deal with the CSS of it all, targeting the right fieldsets and whatnot. Here's an idea: let's use the Bootstrap form component to save us the trouble.

Bootstrap forms are very similar to traditional HTML forms, but they take the fuss out of it. You simply need form groups inside your form, and you're off to the races. Try out my simple form below.

Jumbotrons

Okay, I'll admit it. This made it on the list simply because of it's awesome name. The Jumbotron is big for a reason; it is meant to prominently display the key information of your site. As such, it is perfect for something like a landing page.

In order to make a jumbotron for yourself, you just need a div with the class of jumbotron. Inside, you'll want some reaaaaally big information, so maybe use an h1 with a big display (in bootstrap, you can use the class of display-# to increase the size of a h# tag). After your huge info, maybe some smaller lines and a button or link for the user to follow. Let's look at one in action.

Layout

This is where things can get really cool with Bootstrap. Bootstrap uses a prebuilt grid system so that you don't have to.
What you need to know

Forget about having to build your own grids. Did you forget it? Wow, that was fast. Okay, here's what you need to know. In order to utilise the 12 column system that Bootstrap has built-in, you'll have to place your content into containers, inside of which you'll have to have a div with the class of row, and inside of THAT, divs with the class of col, and another class denoting how many columns that specific block takes up. Sounds confusing, I know. So why don't I show you? Try resizing your browser window for this example, and see what happens to each box.

Column
Column
Column
Column

These columns have two important classes on them - .col-md-6 and .col-lg-3. What this means is that at the prebuilt Bootstrap medium breakpoint, each block occupies 6/12 of Bootstrap's pre-built column layout. Then, at the large Bootstrap breakpoint and above, the columns adhere to the second class of .col-lg-3, which means that each block now takes up 3/12 Bootstrap columns. Oh, and at any point less than the medium breakpoint, the columns automatically stack. Nice! This is a suuuuper convenient way to spread out your content if you want it in a nice, easily divisible grid layout.
