The Joys of Bootstrap

Bootstrap is awesome! It’s an open-source web development toolkit that allows you to take all the fuss out of developing websites and apps. Say goodbye to CSS and Javascipt, and enjoy all the handy pre-built components that Bootstrap has to offer. From navigation bars, to helpful layout tools, and even cards and built in colours, Bootstrap has it all.

Getting started

Are you ready for all the greatness that bootstrap has to offer? Getting your first bootstrap site started is just as easy as you think. All you have to do it go to getbootstrap.com, and you’ll find a few easy ways to get your html document bootstrapped and ready to go. The easiest way, in my opinion, is to copy the starter template down below.

Starter Instructions
  • 1 Head to getbootstrap.com and click "Get started"
  • 2 Scroll down to "Starter template"
  • 3 Copy the HTML in the grey window
  • 4 Paste it in your blank HTML document. You're ready to start!
